# 59. Spiral Matrix II

**Problem Link:** https://leetcode.com/problems/spiral-matrix-ii/

**Difficulty:** Medium

## Problem Statement

Given a positive integer `n`, generate an `n × n` matrix filled with elements from `1` to `n²` in **spiral order**.

---

## Example 1

**Input:**

```text
n = 3
```

**Output:**

```text
[
  [1,2,3],
  [8,9,4],
  [7,6,5]
]
```

---

## Example 2

**Input:**

```text
n = 1
```

**Output:**

```text
[
  [1]
]
```

---

## Constraints

- `1 <= n <= 20`
